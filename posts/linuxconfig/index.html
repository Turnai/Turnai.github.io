1 中文输入法

1.1 安装字体包

直接从windows偷

1.2 安装ibus

sudo apt-get install ibus
ibus-setup

1.3 安装拼音

sudo apt-get install ibus-pinyin
# 或者 sudo apt-get install ibus-libpinyin

之后在ibus图像化界面加入中文拼音即可

2 Clash

使用shellclash 

export url='https://raw.githubusercontent.com/juewuy/ShellClash/master' && sh -c "$(curl -kfsSl $url/install.sh)" && source /etc/profile &> /dev/null

非root用户安装后请额外执行以下命令以读取环境变量：

source ~/.bashrc &> /dev/null

安装完成管理脚本后，执行如下命令以运行管理脚本

clash

3 Firefox ESR

通过添加PPA安装ESR版本

sudo add-apt-repository ppa:mozillateam/ppa
sudo apt-get update
sudo apt install firefox-esr

4 VScode

sudo apt install code

同步账号时出现错误：

dbus systemctl error : GDBus.Error:org.freedesktop.DBus.Error.ServiceUnknown: The name com.example.Interface was not provided by any .service files

解决方法，下个管账号的包：

sudo apt install gnome-keyring

5 QQ

使用第三方QQ ，但缺点是无法发送文件。

使用qq for linux ，很有上古风味。

6 微信

欲尝试优麒麟版本微信 ，但是怕系统信息被修改 ，遂取消。

尝试第三方商店 中的wine-weixin，安装成功，无中文字体。中文字体添加失败，遂卸载。

最终使用统信版本微信 ，只能说勉强能用，同样没有托盘。

7 虚拟机

sudo apt update
sudo apt install -y gcc build-essential
cd ~/Downloads
sudo chmod 700 VMware-Workstation-Full-*.bundle
sudo ./VMware-Workstation-Full-*.bundle

遇到问题：modules编译不能。

尝试教程 

VMWARE_VERSION=workstation-16.2.3
TMP_FOLDER=/tmp/patch-vmware
rm -fdr $TMP_FOLDER
mkdir -p $TMP_FOLDER
cd $TMP_FOLDER
sudo apt install git -y
git clone https://github.com/mkubecek/vmware-host-modules.git
cd $TMP_FOLDER/vmware-host-modules
git checkout $VMWARE_VERSION
git fetch
make
sudo make install
sudo rm /usr/lib/vmware/lib/libz.so.1/libz.so.1
sudo ln -s /lib/x86_64-linux-gnu/libz.so.1 /usr/lib/vmware/lib/libz.so.1/libz.so.1

VMware启动成功，但运行失败。 继续上述教程：

sudo apt install -y openssl
sudo mkdir -p /misc/sign-vmware-modules
sudo chmod 777 /misc/sign-vmware-modules
cd /misc/sign-vmware-modules
openssl req -new -x509 -newkey rsa:2048 -keyout MOK.priv -outform DER -out MOK.der -nodes -days 36500 -subj "/CN=MSI/"
chmod 600 MOK.priv

sudo mokutil --import MOK.der

新建脚本/misc/sign-vmware-modules/signingscript

#!/bin/bash
for modfile in $(modinfo -n vmmon vmnet); do
echo "Signing $modfile"
/usr/src/linux-headers-$(uname -r)/scripts/sign-file sha256 \
/misc/sign-vmware-modules/MOK.priv \
/misc/sign-vmware-modules/MOK.der "$modfile"
done

chmod 700 /misc/sign-vmware-modules/signingscript
sudo /misc/sign-vmware-modules/signingscript

重启系统，但无法进入MOK management。已破防^^。

8 WPS

Libre Office处理不好Word的各种字体和格式，无奈只能用WPS，WPS用了WPS Office 2019 - Multi-Language ——它说没联网，能勉强用用吗？

snap interfaces  | grep -v ".*-$"
sudo snap connect wps-2019-snap:cups-control :cups-control
sudo snap connect wps-2019-snap:alsa :alsa
sudo snap connect wps-2019-snap:pulseaudio :pulseaudio
sudo snap connect wps-2019-snap:removable-media :removable-media

9 Telegram

Telegram无法访问外接硬盘，最后参考上面WPS用snap connections连上removable-media解决了。
